# 邊境據點

這是一個以 Allegro 5 製作、在目前工作區從零建立的開放世界生存建造遊戲原型。

## 核心玩法

- 在大型捲動畫面地圖中自由探索。
- 用 `E` 採集木材、石材、纖維與水晶。
- 以近戰（`Space`）與遠程射擊（滑鼠左鍵）對抗怪物。
- 用 `Q` 切換建造模式，再用滑鼠左鍵放置建築。
- 用 `1`、`2`、`3` 切換 `圍牆`、`砲塔`、`營地`。
- 撐過夜晚怪物襲擊，守住地圖中央的金色核心。

## 已完成內容

- 程序生成的大型地圖，包含水域、草地、原野與深色草區。
- 可採集的樹木、岩石、草叢與水晶礦點，並具備重生時間。
- 背包、等級、經驗值成長、難度提升與核心防守壓力。
- 玩家近戰與遠程戰鬥。
- 多種敵人類型：近戰史萊姆、遠程獵手、重裝巨獸。
- 建築功能：
- 圍牆負責承受傷害。
- 砲塔會自動攻擊附近敵人。
- 營地可在附近持續治療玩家。
- 大字體 HUD、迷你地圖、目標面板、提示訊息、血條與晝夜光照效果。

## 建置說明

此專案需要 Allegro 5 開發函式庫。

範例流程：

```powershell
cmake -S . -B build
cmake --build build
```

如果你的 Allegro 安裝路徑不同，請修改 [CMakeLists.txt](/c:/c++/FIGHTGAME/CMakeLists.txt#L1) 中的 `ALLEGRO_ROOT`。
