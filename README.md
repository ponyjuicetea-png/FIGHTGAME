# 邊境據點

這是一款使用 Allegro 5 製作的 2D 開放世界生存建造遊戲。玩家要在大地圖中探索、採集、建築、防守據點、推進多階段任務，最後擊敗最終魔王。

## 目前內容

- 五大關卡流程：
  - 第一關：採集木材、石材、建立圍牆，並擊敗神社旁的 4 隻黑暗大法師
  - 第二關：建立防線，完成砲塔、營地、黑暗小法師條件並大量清怪
  - 第三關：救出 4 個小孩
  - 第四關：前往損壞的村莊，討伐山匪、黑暗大法師、摧毀藍基地，帶回綠寶石
  - 第五關：前往黑色地點召喚並擊敗最終魔王
- 開放世界地圖探索
- 採集系統：木材、石材、纖維、水晶、菜、肉、魚
- 建築系統：圍牆、砲塔、營地、黑暗小法師
- 多種職業：無業遊民、吸血鬼、伐木工、漁夫、外星人、菁英
- 戰鬥系統：近戰、射擊、毒藥水、炸彈
- 怪物與 Boss 戰
- 商店街系統：炸彈店、超市、工具行
- 飢餓與食物恢復系統
- 晝夜循環與逐日增強的難度
- 永久貨幣紅寶石與神社機制

## 操作方式

- `WASD`：移動
- `滑鼠左鍵`：射擊 / 建造放置
- `E`：採集 / 神社互動
- `Space`：近戰攻擊
- `G`：丟毒藥水
- `H`：釣魚
- `R`：吃食物
- `T`：丟炸彈
- `Q`：切換建造模式
- `1`：圍牆 / 商店選項 1
- `2`：砲塔 / 工具行選項 2
- `3`：營地 / 工具行選項 3
- `4`：黑暗小法師
- `J`：部分職業技能
- `K`：部分職業技能
- `ESC`：開啟暫停選單

## 職業特色

- `1 無業遊民`：沒有特殊能力
- `2 吸血鬼`：每 3 次命中吸血，`J` 回核心
- `3 伐木工`：採集樹石收益更高，雜草與水晶收益更高
- `4 漁夫`：可放船、搭船、船會自動攻擊
- `5 外星人`：`J` 八連砲，`K` 召喚小兵
- `6 菁英`：減傷，毒藥水傷害與持續時間更高

## 商店街

核心附近有一條小街，靠近店面會開啟對應頁面。

- 炸彈店：`1` 用 `20` 水晶換 `10` 個炸彈
- 超市：`1` 用 `20` 木材換 `4` 個肉
- 工具行：
  - `1` 用 `40` 纖維換劍，提升近戰傷害
  - `2` 用 `30` 水晶換盔甲，提升防禦
  - `3` 用 `35` 水晶換藥水強化，提升毒藥傷害

## 神社與永久紅寶石

- 地圖正上方有一座神社
- 必須先打敗 `4` 隻黑暗大法師，才能解除神社封印
- 解鎖後可在神社取得永久紅寶石
- 永久紅寶石不綁定任何單一存檔，而是獨立記錄在 `meta_rubies.txt`

## 存檔系統

- 採用 `100` 格存檔清單
- 存檔檔案格式為 `.txt`
- 存檔位置在 `save_slots/slot001.txt` 到 `save_slots/slot100.txt`
- 標題畫面可用 `L` 進入載入遊戲清單
- 遊戲中按 `ESC` 可開啟暫停選單，再進入存檔 / 讀檔清單
- `F5` 會存到目前存檔格
- `F9` 會讀取目前存檔格
- 遊戲也會自動存到目前使用中的存檔格

## 建置方式

請先安裝 Allegro 5，並確認 CMake 可以找到對應的 include 與 library。

```powershell
cmake -S . -B build
cmake --build build
```

如果 Allegro 安裝位置不是預設路徑，請依照 [CMakeLists.txt](/c:/c++/FIGHTGAME/CMakeLists.txt#L1) 的設定調整 `ALLEGRO_ROOT` 或相關搜尋路徑。
